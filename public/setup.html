CrowdChess Setup

Configure your Lichess API token

Step 1: Create a Lichess Bot Account

You need a Lichess account with Bot access to use CrowdChess.

  1. Register a normal account on Lichess.org
  2. Upgrade it to a Bot account by following the Bot Account Upgrade instructions

Step 2: Generate a Lichess API Token

You need a Personal API Access Token with the following scopes:

  • bot:play - For playing games as a bot
  • challenge:read - For reading incoming challenges
  • challenge:write - For challenging other players

Step 4: Deploy Your Bot

Once you have set up your token, you have two options:

  1. Run locally: Save the .env file to your CrowdChess folder and start the server with npm start
  2. Deploy to Render: Add your token as an environment variable named LICHESS_API_TOKEN in the Render dashboard
